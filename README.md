# caravan-scheduler
小規模組織向けの日産キャラバン（社用車）予約管理Webアプリケーションです。カレンダーベースでの見やすいUIを持ち、簡単に予約の作成・管理ができます。

## 機能
- ユーザー認証（ログイン/ログアウト）
- カレンダーベースの予約表示（月次/週次/日次）
- 予約の作成、編集、キャンセル
- 予約の重複チェック
- ユーザー管理（管理者のみ）

## 技術スタック

### フロントエンド
- React
- Material UI
- FullCalendar.js
- React Router
- Axios

### バックエンド
- Node.js
- Express
- SQLite
- bcrypt（パスワードハッシュ化）
